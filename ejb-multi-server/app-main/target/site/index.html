About Quickstart: ejb-multi-server - app-main

A project that demonstrates multiple applications deployed on different servers. This project builds the EJB and WEB component of the main application and packs it as a deployable EAR archive.

Project Modules

This project has declared the following modules:

Name Description
Quickstart: ejb-multi-server - app-main - ejb A project that demonstrates multiple applications deployed on different servers. This is the main application EJB project.
Quickstart: ejb-multi-server - app-main - web A project that demonstrates multiple applications deployed on different servers. This is the main application web project.
Quickstart: ejb-multi-server - app-main - ear A project that demonstrates multiple applications deployed on different servers. This is the main application EAR project. Includes the EJB and web application together with the necessary API libraries of the sub applications app-one and app-two.
